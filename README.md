# ImpactDAO

A decentralized autonomous organization (DAO) for managing impact-driven projects built on Stacks blockchain using Clarity smart contracts.

## Overview
ImpactDAO enables community-driven governance of social impact projects through:
- Proposal creation and voting
- Project funding allocation
- Impact measurement and tracking
- Token-based membership and governance

## Contracts
- `impact-dao.clar` - Core DAO functionality and governance
- `impact-token.clar` - DAO membership and governance token
- `project-registry.clar` - Project registration and management
